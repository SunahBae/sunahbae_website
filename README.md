# Faithful Folio Builder

## 프로젝트 정보

포트폴리오 웹사이트 프로젝트입니다.

## 코드 수정 방법

### 로컬 개발

Node.js & npm이 설치되어 있어야 합니다 - [nvm으로 설치](https://github.com/nvm-sh/nvm#installing-and-updating)

다음 단계를 따르세요:

```sh
# Step 1: 저장소 클론
git clone <YOUR_GIT_URL>

# Step 2: 프로젝트 디렉토리로 이동
cd <YOUR_PROJECT_NAME>

# Step 3: 의존성 설치
npm i

# Step 4: 개발 서버 시작
npm run dev
```

## 사용 기술

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## 배포

```sh
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.
